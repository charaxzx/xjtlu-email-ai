使用指南

1. 配置 AI 服务

点击顶部的“配置”，填写 AI Base URL、API Key 和模型名称。保存后立即生效。

"ai": { "base_url": "https://api.openai.com/v1", "api_key": "sk-...", "model": "gpt-4o-mini" }

2. 导入 Cookie

  1. 在 Chrome 或 Edge 中安装 Cookie-Editor。
  2. 登录 mail.xjtlu.edu.cn。
  3. 使用 Export as JSON 导出 Cookie。
  4. 把内容粘贴到配置弹窗中的 Cookie 字段。

如果 Cookie 失效，可重新导出后覆盖保存。

3. 使用方式

  • 关键词留空时，程序直接读取最近邮件。
  • 输入总结指令后，系统会搜索、提取正文并生成总结。
  • 查询结果和搜索历史会保存在本地浏览器存储中。

自定义主标语

可自由选择展示方式；颜色、字号、分段均可自定义（HTML 模式）。内容仅保存在本机浏览器。

HTML 会移除 script/iframe 等危险标签；仍可用内联 style、class。主题色变量：var(--accent)、var(--text-strong)。
